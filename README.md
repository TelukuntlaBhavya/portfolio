# 🌐 Bhavya's Portfolio

🚀 Live Demo: https://jazzy-squirrel-a3f5cc.netlify.app/

## 📌 About

This is my personal portfolio website built using modern web technologies to showcase my projects, skills, and experience in software development and data science.

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 Tailwind CSS
* 🎭 Framer Motion

## ✨ Features

* Responsive design (mobile + desktop)
* Smooth animations and transitions
* Project showcase section
* Contact section

## 📂 Project Structure

```
portfolio/
├── public/
├── src/
├── index.html
├── package.json
└── vite.config.js
```

## 🚀 Getting Started

### Clone the repository

```
git clone https://github.com/TelukuntlaBhavya/portfolio.git
```

### Navigate to project

```
cd portfolio
```

### Install dependencies

```
npm install
```

### Run locally

```
npm run dev
```

### Build for production

```
npm run build
```
